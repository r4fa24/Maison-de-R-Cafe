# Maison de R Café ☕🥐
A premium, responsive web landing page designed for a luxury café experience.

## 🚀 Project Overview
Maison de R is a front-end project that captures the "timeless elegance" of a Parisian café. It features a curated digital menu, an interactive gallery of moments, and a localized contact section for a Downtown Dubai establishment.

## ✨ Key Features
* **Responsive Architecture:** Built with a mobile-first mindset using a flexible viewport meta tag to ensure a seamless experience across all devices.
* **Curated Digital Menu:** Displays signature blends like the "Macaron Matcha" and "vAleRAF Rose Mocha" using semantic HTML sections.
* **Integrated Gallery:** Features a specialized gallery section showcasing high-quality assets to drive user engagement.
* **Modern Branding:** Focuses on clean typography and a sophisticated color palette to match the "Maison de R" brand identity.

## 🛠️ Tech Stack
* **Languages:** HTML5, CSS3
* **Tools:** Google Maps API Integration
